  5. 在git命令行中设置用户名和邮箱

  6. 在Powershell（cmd）输入 ssh-keygen -t rsa -C “your email”回车，一直回车

  8. 复制id_rsa.pub文件中的内容，在GitHub设置中新建ssh key并填入生成公钥内容
