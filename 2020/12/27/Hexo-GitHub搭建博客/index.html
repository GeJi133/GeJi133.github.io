本地无密码连接GitHub

  1. 在git命令行中设置用户名和邮箱

  2. 在Powershell（cmd）输入 ssh-keygen -t rsa -C "your email"回车，一直回车

  3. 在输出信息中找到生产密钥地址，一般是在.ssh文件夹，id_rsa.pub公钥文件，id_rsa是私钥文件。

  4. 复制id_rsa.pub文件中的内容，在GitHub设置中新建ssh key并填入生成公钥内容
